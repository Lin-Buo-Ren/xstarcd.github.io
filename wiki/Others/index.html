  • IPHONE接听电话时，无法滑动解锁BUG
    • http://bbs.app111.com/thread-345085-1-1.html 
      很多FY都遇到过IPHONE接听电话时，无法滑动解锁的尴尬局面，有人发帖说是因为电容屏的物理特性造成的，但经过测试表明，我的结论是属于IPNHE的BUG

测试方法：
1、拨入电话先挂断电话。
2、IPHONE接听时靠近感光探头使屏幕处于黑屏状态（可以把手机扣放在桌面）
3、拨入电话挂断后，让IPHONE仍保持黑屏状态，不要对IPINONE做任何操作，持续时间约1分钟以上（相当于自动锁屏时长）。
4、再次拨打IPHONE，无法解锁现象出现！！！！

有人说上述测试是非常规操作，不符合正常接听电话的状态，因为谁那么傻接电话时会把电话扣在桌面上？！那么我们采用正常接听状态下测试：
1、用其他电话拨通你的IPHONE
2、你接听电话，IPHONE这时自然是要放在你的耳边，屏幕处于黑屏状态。
3、其他电话挂断通话。
4、这时，你会习惯性的把IPHNOE直接放入口袋（注意关键点出现：对方挂线后，IPHONE离开你的耳朵到屏幕重新亮起来，时间约2、3秒钟，而你恰好在屏幕还没有亮起来之前，就把手机放进了口袋或包里，这时IPHONE肯定仍处于黑屏状态，如果这个环节屏幕亮起来的话，无法解锁的现象就不会出现）
5、过1分钟以后再拨打IPHONE（这也是关键点，这个时间是IPHONE启动屏幕自动锁屏的时间），无法解锁的现象出现！！！！！

所以为了避免出现无法解锁的现象，在你接电话且对方先挂线的情况下，不要立即把IPHONE放进口袋或包里，而是让屏幕亮起来后再放进口袋或包里，这样我想基本能告别下次接听电话无法解锁的尴尬局面。 也可以按一下电源键

这里要感谢另外一个FY--wchdsb4 ，因为是他首先提出无法滑动解锁是BUG，而非硬件物理特性导致，加上我这个人比较轴，所以通过反复做上述实验，验证确属BUG。所以说实践出真知，反对伪科学！

  • 特殊的链接：打电话，短信，email；iPhone 和Android应用
    • http://qing.blog.sina.com.cn/tj/6afc58b033000459.html
    • 电话:IPhone的写法: <a href="callto:[phone_number]">phone_number</a>,例子：
      • <a href="callto:12345678">12345678</a>:12345678
    • 电话:Android的写法: <a href="wtai://wp/mc;[phone_number]">phone_number</a>,例子：
      • <a href="wtai://wp/mc;12345678">12345678</a>: 12345678
      • 在电话号码前面可以加上+（加号）表示国际号码。如：
        • <a href="wtai://wp/mc;+12345678">+12345678</a>:+12345678
    • 短信:如果是需要调用短信的接口，可以将链接写成下面的格式：
      • sms:<phone_number>[,<phone-number>]*[?body=<message_body>]，如：
        • <a href="sms:12345678">给 12345678 发短信</a>:给 12345678 发短信
        • <a href="sms:12345678?body=hello">给 12345678 发送内容为"hello"的短信</a>:给 12345678 发送内容为"hello"的短信
        • <a href="sms:12345678,98765432?body=hello">给12345678和98765432 发送内容为"hello"的短信</a>:给12345678和98765432 发送内容为"hello"的短信

  • 魔方入门玩法：http://www.rubik.com.cn/beginner.htm
    • 第一步：对好第一面十字
    • 第二步：对好第一面加上四个侧面的T字形
    • 第三步：处理第二快去的四个棱色块，对好前两层(远切回回，接孩子回家:URU'R' U'F'UF)
    • 第四步：在新的黄色顶面画十字(FRUR' U'F')
    • 第五步：调整顶层角色块的朝向，对好顶层黄色面
      • (侧面黄色顺时针方向)小鱼1:R'U'RU' R'U'2R
      • (侧面黄色逆时针方向)小鱼2:FUF'U FU2F'
      • 如果有2个黄色不在顶面，让一个黄色冲左后角，用小鱼1
      • 如果有4个黄色不在顶面，让一个两个黄色冲左边，用小鱼1
    • 第六步：调整顶层角色块顺序:让顶层黄色面朝着自己，R2D2 R'U'RD2 R'UR'，让黄色面重新朝上，旋转顶层，对齐四角颜色
    • 第七步：调整顶层棱色块顺序，将魔方最后还原
      • 顶层棱色逆时针调换：好的一面朝正面，小鱼1->小鱼2
      • 顶层棱色顺时针调换：好的一面朝右面，小鱼2->小鱼1

暑假马上就要来临，与其让孩子成天观看一些动画片或者玩一些无聊的游戏，不如带着孩子一起看看这些励志电影吧，让孩子在暑假放松玩耍的同时，还能对孩子的成长有正面的影响，引导孩子正面的思考,对培养孩子的良好性格也会起到潜移默化的作用。

  2. 《风雨哈佛路》，一个最贫困的哈佛女孩从不退缩的奋斗，看完让人满身温暖，央视曾经播过。
  3. 《听见天堂》，一个热爱电影的盲童选择了用耳朵代替眼睛，去记录他生活的点点滴滴。
  4. 《卡特教练》，一个篮球教练率领一群看不到出路的孩子们为梦想作战。
  5. 《追梦女孩》，碧昂斯主演，3个黑人女孩踏上充满未知的星途最后获得成功的故事。
  6. 《阳光小美女》，充满亲情温暖的励志片。
  7. 《闻香识女人》，阿尔帕·西诺主演，里面的一曲《一步之遥》探戈舞曲奏响生命的热情。

  11. 《当幸福来敲门》，威尔·史密斯难得的非商业电影，和儿子一起出演，很有噱头，而且很励志，很好看。

  14. 《永不妥协》，茱莉亚·罗伯茨借此片获得影后，十分真诚的励志电影。
  15. 《死亡诗社》，罗宾·威廉姆斯主演的经典励志电影，还是扮演一个春风化雨的老师，绝对推荐。

  • 办证指南:http://www.cdjg.gov.cn/Html/News/20131213/content_842_4928.html
  • 一、所需资料
    • 1、《机动车驾驶证申请表》；
    • 2、县级或部队团级以上医疗机构出具的《机动车驾驶人身体条件证明》，属于申请残疾人专用小型自动挡载客汽车的，应当提交经省级卫生主管部门指定的专门医疗机构出具的《机动车驾驶人身体条件证明》；

    • 4、原机动车驾驶证原件；
    • 5、近期一寸免冠白底彩色照片三张；
  • 二、办理时限：一个工作日发证
  • 三、办理流程：综合服务大厅提交资料、缴费、领取驾驶证
  • 四、办理地点：成都车管所及所属辖区交警分局和区（市）县交警大队
  • 五、其它说明
    • 1、持有A1、A2、A3、B1、B2驾驶证的驾驶人，应当在每个记分周期结束后三十日内到公安机关交通管理部门接受审验。但在一个记分周期内没有记分记录的，免予本记分周期审验；
    • 2、持有A1、A2、A3、B1、B2驾驶证以外准驾车型驾驶证的驾驶人，发生交通事故造成人员死亡承担同等以上责任未被吊销机动车驾驶证的，应当在本记分周期结束后三十日内到公安机关交通管理部门接受审验；
    • 3、申请前核查申请人不具有记满12分、逾期未审验、被扣押、扣留、暂扣、注销、吊销或者撤销机动车驾驶证的情形；
    • 4、属于驾驶证有效期满换证的，还应当核查不具有道路交通安全违法行为、交通事故未处理完毕的情形；
    • 5、受理前先行核查申请人是否存在有机动车违法未处理、未参加定期检验或未办理报废注销的情况，应当告知申请人先行处理，再办理业务；若所持驾驶证有效期满截止日期在30日内的，申请人可作出书面承诺后办理有关业务；
    • 6、如有任何疑问请拨打车管所咨询电话028-82001010.
